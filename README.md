# Developer & User Guide: Testing the MRI ERP System

This guide provides step-by-step instructions on how to initialize the database and test the built-in PL/SQL logic for GST calculations, Inventory, and MRP.

---

## 1. Initial Setup
Ensure you have executed the `MRIERP_PROJECT.sql` script. This will:
1. Create the `ERP` user.
2. Build all 20+ tables.
3. Compile the Packages and Triggers.
4. Insert sample MRI components and Vendor data.
